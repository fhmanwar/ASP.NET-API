# ASP.NET-API

to use url:

GET ALL :

    https://localhost:44344/api/suppliers/

GET ID :

    https://localhost:44344/api/suppliers/{id}

POST :

    https://localhost:44344/api/suppliers/

    raw body:
    {
        "Name": "asd"
    }

PUT :

    https://localhost:44344/api/suppliers/{id}

    raw body:
    {
        "Id": {id},
        "Name": "{nama}"
    }

DELETE : 

    https://localhost:44344/api/suppliers/{id}
